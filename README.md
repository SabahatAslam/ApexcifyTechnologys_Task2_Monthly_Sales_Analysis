## Task 2 – Analyzing Monthly Sales of a Company 📊

### 📌 Introduction
✨The project focuses on analyzing a customized monthly sales dataset to uncover trends and generate meaningful business insights.
 
### The project includes:

- `📓 Jupyter Notebook (.ipynb`)
- `🐍 Python Script (.py`)
- `🌐 HTML Export (.html`)
- `📁 Dataset (.csv`)

You can view the ipynb file here in colab:

### 📝Customized Dataset

The dataset contains **12 rows and 2 columns**. 
To simulate real-world scenarios, **seasonal peaks and dips** were intentionally added (e.g., peak holiday season and post holiday dip). This helps in learning seasonal fluctuations or marketing/promotion impacts for business-oriented data analysis.

---

### 📦 Required Libraries

Make sure you have **Python 3.10+** and the following libraries installed:

- `pandas`
- `numpy`
- `matplotlib`
- `seaborn`

*(See `requirements.txt` for the complete list)*

---

### 🔑 Key Concepts
The following important data analysis concepts are used in this project:

* **Data Handling:**
  `read_csv()`, `DataFrame`

* **Data Visualization:**
  `plot()`, `bar()`, `xlabel()`, `ylabel()`, `title()`, `xticks()`, `show()`

---
### ⚙️Project Objectives
- Organize and explore sales data for all 12 months
- Visualize sales trends using line charts with Matplotlib
- Compare monthly sales performance using bar charts
- Identify highest and lowest sales months
- Analyze overall sales trends and seasonal patterns
- Highlight monthly fluctuations and growth patterns
- Present insights through clear and labeled visualizations
- Help interpret business performance over a yearly period
